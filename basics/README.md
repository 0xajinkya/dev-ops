## Kubernetes & MiniKube Commands

# Apply YAML
```bash
kubectl apply -f file_name.yaml
```

# Get Pods
```bash
kubectl get pods
```

# Get Detailed Info About Pods
```bash
kubectl get pods -o wide
```

# Get Services
```bash
kubectl get service
```

# Get Deps
```bash
kubectl get deployment
```

# Get ConfigMaps
```bash
kubectl get configmap
```

# Get Secrets
```bash
kubectl get secret
```

# Minikube IP
```bash
minukube ip
```

# Run A Service
```bash
minikube service webapp-service
```
